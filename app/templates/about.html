About Flask Blog

Flask Blog is a simple blogging platform built with Flask. It allows users to create and share their thoughts with the world.

Features:

  • User authentication system with registration and login
  • Role-based access control (admin and regular users)
  • Create, read, update, and delete blog posts
  • Admin dashboard for managing users and all posts
  • Responsive design for great experience on all devices

This application was built as a demonstration of Flask capabilities and best practices.
